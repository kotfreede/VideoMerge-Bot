# VideoMerge-Bot
This is very simple Telegram Videos Merge Bot by [@KOTFREEDE](https://t.me/kotfreede). Using FFmpeg for Merging Videos.

## Features:
- Merge Multiple Videos.
- Permanent Custom Thumbnail Support.
- Broadcast To All Started Users.
- 2GB+ Files will Upload to Streamtape.
- Others check by your own feeling lazy to describe.
